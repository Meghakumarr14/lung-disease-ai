# 🫁 AI-Powered Lung Disease Detection System

🚀 **Live Demo:** https://lung-disease-ai.vercel.app
🔗 **Backend API:** https://meghakumarr14-lung-disease-backend.hf.space

An end-to-end **AI-based medical imaging system** that analyzes chest X-ray images to detect lung diseases using a **SENet154 deep learning model**. The system also provides **Grad-CAM visual explanations**, an **AI medical chatbot**, and **automated clinical report generation**.

---

# 🧠 Key Features

✔ Lung disease prediction from chest X-ray images
✔ Top disease probability predictions
✔ **Grad-CAM heatmap visualization** for explainable AI
✔ **Automated medical report generation (PDF)**
✔ **Interactive medical chatbot**
✔ Modern **React web interface**
✔ Cloud deployment with **Hugging Face + Vercel**

---

# 🫁 Detectable Conditions

The model can classify chest X-ray images into the following categories:

* Normal
* Pneumonia
* Tuberculosis
* COVID-19
* Lung Opacity

---

# 🧪 AI Model

Model Architecture: **SENet154 (Squeeze-and-Excitation Network)**

Why SENet?

* Improves feature representation
* Better channel attention
* High performance in medical image classification

Frameworks used:

* PyTorch
* TIMM model library

---

# 🔍 Explainable AI (Grad-CAM)

The system uses **Gradient-weighted Class Activation Mapping (Grad-CAM)** to highlight important regions of the X-ray image that influenced the model's decision.

This improves **model transparency and clinical interpretability**.

---

# 🏗️ System Architecture

Frontend: React.js (Vercel)
Backend: FastAPI (Hugging Face Spaces)
Model: PyTorch SENet154

User Uploads X-ray

↓

↓
Results + PDF Report
